## What's New

MistratonReview is live. Install the GitHub App on any repository and get instant AI-powered code reviews on every pull request, powered by Mistral AI.

### Core Features

- **GitHub App installation** with one-click setup via OAuth flow
- **Automatic PR review** triggered by webhooks on PR open and update events
- **Mistral AI analysis** using `mistral-large-latest` with a senior reviewer system prompt
- **Inline code comments** posted on exact diff lines in the pull request
- **Summary review comment** with an overview of all findings and their severity breakdown
- **Severity levels** - critical, warning, and suggestion categories for every comment
- **Landing page** with "Install on GitHub" call-to-action and feature highlights

### How It Works

When a pull request is opened or updated, GitHub sends a webhook to MistratonReview. The app fetches the PR diff, sends it to Mistral AI with a structured prompt, and posts the resulting review comments directly back to GitHub as inline annotations on the changed lines.

The review summary comment appears at the top of the PR timeline and breaks down findings by severity so reviewers can prioritize critical issues first.
