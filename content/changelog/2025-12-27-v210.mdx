## What's New

v2.1 adds a dashboard for review metrics, a setup page that guides users through GitHub App installation, a public changelog, and a refreshed sunset theme.

### Features

- **Review metrics dashboard** - See total reviews performed, comment counts by severity, and recent PR activity across all installed repositories
- **Setup page** - A guided flow shown immediately after GitHub App installation that confirms webhook configuration and runs a test review
- **Public changelog system** - This very page, built with MDX-based content, a timeline UI, and draft/published status per entry
- **Sunset theme** - A warm orange-to-red color palette applied across the app and landing page

### Dashboard Metrics

The dashboard shows:

- Total pull requests reviewed (all time and last 30 days)
- Breakdown of comments by severity (critical, warning, suggestion)
- List of recently reviewed repositories with links to the latest PR review
- Estimated time saved based on average human review duration

### Setup Page

After installing MistratonReview on a GitHub organization or repository, users are redirected to the setup page. The page verifies that the webhook is reachable, displays the installation ID, and offers a one-click test review against a sample diff to confirm everything is working before the first real PR arrives.
